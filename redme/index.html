Wonderlust - Hotel Reservation System

Project Overview

Wonderlust is a comprehensive hotel reservation system designed to simplify the process of booking accommodations. It provides users with an intuitive interface to search, compare, and reserve hotel rooms seamlessly.

Features

  • Search hotels by location, price, and amenities.
  • Real-time availability and booking confirmation.
  • User-friendly interface for both customers and administrators.
  • Secure payment gateway integration.
  • Customer reviews and ratings for hotels.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Payment Integration: Stripe/PayPal
